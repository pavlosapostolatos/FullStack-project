# a generic social media website using React frontend node.js and MySql backend and Database
## Installation Instructions

To install you just need to do the following:
- import the .sql dump file into your MySql workbench
- Install [NodeJs and npm](https://nodejs.org/en/)
- Navigate to the react folder which contains the frontend and install the node-modules by running `npm install` 
- Navigate to the sqlserver backend folder and install the node-modules by running `npm install`

## Execution Instructions

To run the application you just need to:
- Run the backend by going to the `sqlserver` folder and then run the command `node index.js`
- Run the frontend by going to the `react` folder and then run the command `npm start`
- Navigate to https://localhost:3000 .
- The first time you use the app with a clean database, look at the user MySql table for login credentials
- Use the app!!!
